# 05_supplier_sees_requisites.png

Capture: страница /supplier/documents в кабинете поставщика.

На скриншоте должно быть видно:
- Название ресторана + ИНН
- Блок «Реквизиты ресторана» с полями (юр. название, ИНН, ОГРН, адреса, контакты и т.д.)
- Список документов

Ресторан должен быть в статусе «Договор принят» (linked) — тогда реквизиты отображаются.
